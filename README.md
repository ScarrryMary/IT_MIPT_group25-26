# IT_MIPT_group25-26

Всем привет! Здесь будут находиться семинары и домашки для группы Буйловой М.А. по информатике (МФТИ, ФЭФМ) в 25/26 учебном году. Stay tuned!

1 семестр:
Семинар 2: Строки, массивы, срезы, работа с файлами  
Семинар 3: форматирование строк, обработка текста  
Семинар 4: работа с Git, рекурсия, Алгоритм быстрого возведения в степень, числа Фибоначчи, NumPy-начало  
Семинар 5: Построение графиков, множества и словари  
Семинар 6: Одномерное динамическое программирование, числа Фибоначчи с мемоизацией, задача о кузнечике, бинарный поиск  
Семинар 7: лямбда-функции, классы  
Семинар 8: Двумерное динамическое программирование (Вычисление наибольшей возрастающей подпоследовательности, Наибольшая общая подпоследовательность, Задача о разбиении строки, Черепашка, Расстояние Левенштейна)  
Семинар 9: Закрепление ООП (Стек, очередь, дек), Функции от произвольного количества аргументов, Декораторы  
Семинар 10: Двусвязный список, Асимптотика алгоритмов, Сортировки (Случайная сортировка, Сортировка пузырьком, Сортировка выбором, Сортировка вставками, Быстрая сортировка (сортировка Хоара), Сортировка слиянием, Сортировка подсчетом, Поразрядная сортировка)  
Семинар 11: Введение в деревья, двоичные деревья, куча, сортировка кучей, Хеширование (Хеш-таблицы, методы разрешения коллизий), Алгоритм Рабина-Карпа
